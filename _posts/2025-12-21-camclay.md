## 1. Tham S·ªë M√¥ H√¨nh Modified Cam-Clay

### 1.1 C√°c Th√¥ng S·ªë C∆° B·∫£n

M√¥ h√¨nh MCC ƒë∆∞·ª£c ki·ªÉm so√°t b·ªüi 7 th√¥ng s·ªë ƒë·ªãa k·ªπ thu·∫≠t ch√≠nh:

| Th√¥ng S·ªë | K√Ω Hi·ªáu | M√¥ T·∫£ | ƒê∆°n V·ªã | Gi√° Tr·ªã M·∫´u |
|----------|---------|-------|--------|-----------|
| √Åp su·∫•t ti·ªÅn c·ªë k·∫øt ban ƒë·∫ßu | $p_c$ | Preconsolidation pressure | kPa | 150 |
| √Åp su·∫•t g·ªëi t·ª±a ban ƒë·∫ßu | $p_0$ | Initial confining pressure | kPa | 150 |
| Tham s·ªë g√≥c ma s√°t tr·∫°ng th√°i t·ªõi h·∫°n | $M$ | Critical friction parameter | - | 0.95 |
| Ch·ªâ s·ªë n√©n | $\lambda$ | Compression index | - | 0.2 |
| Ch·ªâ s·ªë n√©n/n·ªü l·∫°i | $\kappa$ | Recompression index | - | 0.04 |
| Tham s·ªë ƒë∆∞·ªùng NCL | $N$ | Normal Consolidation Line parameter | - | 2.5 |
| T·ª∑ l·ªá Poisson | $\nu$ | Poisson's ratio | - | 0.15 |

### 1.2 C√°c Kh√°i Ni·ªám Ch√≠nh

**·ª®ng Su·∫•t H·ªØu Hi·ªáu ($p'$ ho·∫∑c $p$)**  
·ª®ng su·∫•t h·ªØu hi·ªáu l√† s·ª± k·∫øt h·ª£p c·ªßa ·ª©ng su·∫•t t·ªïng v√† √°p l·ª±c n∆∞·ªõc l·ªó r·ªóng:
$$p' = \frac{\sigma_1' + \sigma_2' + \sigma_3'}{3}$$

**·ª®ng Su·∫•t L·ªách ($q$)**  
ƒê·ªô l·ªách ·ª©ng su·∫•t ch√≠nh ƒë∆∞·ª£c ƒë·ªãnh nghƒ©a trong th·ª≠ nghi·ªám triaxial:
$$q = \sigma_1' - \sigma_3'$$

**H·ªá S·ªë R·ªóng ($e$) v√† Th·ªÉ T√≠ch Ri√™ng ($V$)**  
$$e = V - 1.0$$
$$V = N - \lambda \ln(p_c) + \kappa \ln\left(\frac{p_c}{p_0}\right)$$

**M·∫∑t Ch·∫£y Modified Cam-Clay**  
B·ªÅ m·∫∑t ch·∫£y trong kh√¥ng gian $p'-q$ ƒë∆∞·ª£c m√¥ t·∫£ b·ªüi:
$$f = q^2 - M^2 p'(p_c - p') = 0$$

### 2.4 V√≤ng L·∫∑p ·ª®ng Su·∫•t-Bi·∫øn D·∫°ng (Ch√≠nh)

V√≤ng l·∫∑p ch√≠nh th·ª±c hi·ªán 7500 b∆∞·ªõc t√≠nh to√°n, m·ªói b∆∞·ªõc bao g·ªìm:

1. **T√≠nh M√¥-ƒëun ƒê√†n H·ªìi**
   $$K = \frac{V \cdot p}{Œ∫}, \quad G = \frac{3K(1-2ŒΩ)}{2(1+ŒΩ)}$$

2. **Ma Tr·∫≠n ƒê·ªô C·ª©ng ƒê√†n H·ªìi (6√ó6)**
   - ƒê∆∞·ªùng ch√©o ch√≠nh: $K + \frac{4G}{3}$
   - Ngo√†i ƒë∆∞·ªùng ch√©o: $K - \frac{2G}{3}$
   - Ph·∫ßn c·∫Øt: $G$

3. **Ki·ªÉm Tra Tr·∫°ng Th√°i Ch·∫£y**
   - N·∫øu $f < 0$: S·ª≠ d·ª•ng ma tr·∫≠n ƒë√†n h·ªìi
   - N·∫øu $f ‚â• 0$: S·ª≠ d·ª•ng ma tr·∫≠n ƒë√†n d·∫ªo

4. **T√≠nh Bi·∫øn D·∫°ng T∆∞∆°ng ·ª®ng**
   - Triaxial Drained: $d\varepsilon_1 = de$, $d\varepsilon_2 = d\varepsilon_3$ t·ª´ ƒëi·ªÅu ki·ªán bi√™n
   - Triaxial Undrained: $d\varepsilon_1 = de$, $d\varepsilon_2 = d\varepsilon_3 = -\frac{de}{2}$

5. **C·∫≠p Nh·∫≠t ·ª®ng Su·∫•t**
   $$d\sigma = \mathbf{D} \cdot d\varepsilon$$

```julia
for a in 2:iteration
    # Bulk and shear moduli
    K = V * p[a-1] / Œ∫
    G = (3 * K * (1 - 2 * ŒΩ)) / (2 * (1 + ŒΩ))
    
    # Update preconsolidation pressure on yield surface
    if yield == 0
        pc_current = (q[a-1]^2 / M^2 + p[a-1]^2) / p[a-1]
    else
        pc_current = cp
    end
    
    # Elastic stiffness matrix (6x6)
    De = zeros(6, 6)
    
    # Fill elastic stiffness
    for m in 1:3
        for n in 1:3
            if m == n
                De[m, n] = K + 4 * G / 3
            else
                De[m, n] = K - 2 * G / 3
            end
        end
    end
    
    for m in 4:6
        De[m, m] = G
    end
    
    # Determine stiffness matrix (elastic or plastic)
    if yield < 0
        # Elastic stiffness
        D = De
    else
        # Elastoplastic stiffness - compute gradients
        dfds = zeros(6)
        dfdep = zeros(6)
        
        for m in 1:3
            dfds[m] = (2 * p[a-1] - pc_current) / 3 + 3 * (S[m] - p[a-1]) / M^2
            dfdep[m] = (-p[a-1]) * pc_current * (1 + e‚ÇÄ) / (Œª - Œ∫) * 1
        end
        
        # Elastoplastic matrix
        numerator = De * dfds * dfds' * De
        denominator = -(dfdep' * dfds)[1] + (dfds' * De * dfds)[1]
        
        if abs(denominator) > 1e-12
            D = De - numerator ./ denominator
        else
            D = De
        end
    end
    
    # Strain increment based on analysis type
    dStrain = zeros(6)
    if analysis == 1
        # Triaxial Drained
        dStrain[1] = de
        if abs(D[2,2] + D[2,3]) > 1e-12
            dStrain[2] = -1 * D[2,1] / (D[2,2] + D[2,3]) * de
        else
            dStrain[2] = -de / 2
        end
        if abs(D[3,2] + D[3,3]) > 1e-12
            dStrain[3] = -1 * D[3,1] / (D[3,2] + D[3,3]) * de
        else
            dStrain[3] = -de / 2
        end
    else
        # Triaxial Undrained
        dStrain[1] = de
        dStrain[2] = -de / 2
        dStrain[3] = -de / 2
    end
    
    # Stress increment and update
    dS = D * dStrain
    S = S + dS
    strain = strain + dStrain
    
    # Update invariants and state variables
    depsV = dStrain[1] + dStrain[2] + dStrain[3]
    depsD = (2 / 3) * (dStrain[1] - dStrain[3])
    
    V = N - Œª * log(pc_current) + Œ∫ * log(pc_current / p[a-1])
    
    p[a] = (S[1] + S[2] + S[3]) / 3
    q[a] = S[1] - S[3]
    u[a] = p‚ÇÄ + q[a] / 3 - p[a]
    
    void[a] = V - 1.0
    epsV[a] = epsV[a-1] + depsV
    epsD[a] = epsD[a-1] + depsD
    
    # Update yield surface
    if yield < 0
        yield = q[a]^2 + M^2 * p[a]^2 - M^2 * p[a] * pc_current
    else
        yield = 0
    end
end
```

## 3. Ph√¢n T√≠ch K·∫øt Qu·∫£ M√¥ Ph·ªèng

### 3.1 ƒê·∫∑c T√≠nh M·∫∑t Ch·∫£y (Yield Surface)

**ƒê∆∞·ªùng C·ªë K·∫øt B√¨nh Th∆∞·ªùng (Normal Consolidation Line - NCL)**  
Trong kh√¥ng gian e-ln(p'), ƒë·∫•t s√©t c·ªë k·∫øt b√¨nh th∆∞·ªùng s·∫Ω n·∫±m tr√™n:
$$e = N - \lambda \ln(p')$$

**ƒê∆∞·ªùng Tr·∫°ng Th√°i T·ªõi H·∫°n (Critical State Line - CSL)**  
Khi ƒë·∫•t ch·∫£y h·∫øt s·ª©c (critical state), n√≥ tu√¢n theo:
$$e = \Gamma - \lambda \ln(p')$$

**M·∫∑t Ch·∫£y Modified Cam-Clay**  
Trong kh√¥ng gian $p'-q$, b·ªÅ m·∫∑t ch·∫£y c√≥ d·∫°ng ellipse:
$$q = M \cdot p' \quad \text{(Critical State Line slope)}$$
$$f = q^2 - M^2 p'(p_c - p') = 0$$

```julia
# Initial Yield Surface

qy_ini = sqrt.(M^2 .* (cp .* p_ini_yield .- p_ini_yield.^2))

# Normal Consolidation Line (NCL)

eNCL = N .- Œª .* log.(pNCL) .- 1

# Final Yield Surface
p_fyield = collect(0:pc/100:pc)
q_fyield = M .* p_fyield
qyf = sqrt.(M^2 .* (pc .* p_fyield .- p_fyield.^2))

# Critical State Line (CSL)
pCSL = pNCL
Gamma = 1 + void[end] + Œª * log(p[end])
eCSL = Gamma .- Œª .* log.(pCSL) .- 1
```
