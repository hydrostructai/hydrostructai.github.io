# hydrostructai.github.io

Các ứng dụng Web cho Kỹ thuật Xây dựng. Trang web này được xây dựng bằng Jekyll và theme Minimal Mistakes.

## Nội dung

Trang web này chứa các bài viết blog và các ứng dụng web tương tác (Web Apps) tập trung vào:
* Kỹ thuật Xây dựng (Civil Engineering)
* Địa kỹ thuật (Geotechnical Engineering)
* Thủy lợi (Hydraulics)
* Trí tuệ nhân tạo (AI) trong Xây dựng

## Giấy phép

Nội dung của trang web này được cấp phép theo giấy phép MIT.
